## 🔀 Git 컨벤션

### 브랜치 네이밍

```
chapter{번호}/{이름}
```

**예시:**

- `chapter01/홍길동`
- `chapter05/seho`

### 커밋 메시지

```
[chapterXX/이름] 커밋 내용
```

**예시:**

- `[chapter01/이세호] 요약 및 정리 작성`
- `[chapter01/홍길동] 딥다이브 문서 추가`
- `[chapter03/seho] 오타 수정`

### PR 컨벤션

**제목:**

```
[ChapterXX] 이름
```

**예시:**

- `[Chapter01] 홍길동`
- `[Chapter05] 이세호`

**본문 템플릿:**

- PR 본문은 비워두셔도 좋습니다.

## 🎯 발표 방식

1. **메인 발표자**: 매주 랜덤 1명 선정
   - 모든 참여자가 1회씩 발표 후 → 다시 전체 랜덤 순환
2. **메인 발표자**: 딥다이브 문서 기반 발표
3. **일반 참여자**: 가장 인상 깊었던 인사이트 1개 공유
